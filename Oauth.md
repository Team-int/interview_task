# 디스코드 API 이해도

![image](https://github.com/team-int/interview_task/raw/backend/img/oauth.png)

### 조건

*   디스코드의 Oauth API를 사용하여 로그인 기능을 만듭니다.
*   사용자의 정보를 암호화 해야합니다.

### 방법

1.  해당 레포지토리를 포크합니다.
2.  포크한 레포지토리에 코딩을 합니다.
3.  코딩이 끝난 후 개인 레포지토리에 해당 소스코드를 올려 제출합니다.
4.  제출 후 검토가 끝날 시 해당 레포지토리를 삭제해도 됩니다.
